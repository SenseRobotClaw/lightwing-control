# 使用说明（给用户）

你可以直接这样说：

## 基础灯控
- 开灯
- 关灯
- 亮度调到最亮
- 亮度调暗一点
- 色温调到最小值
- 色温调暖一点
- 切到专注阅读模式
- 切到标准照明模式

## 语音朗读（TTS）
- 让台灯说"你好"
- 帮我朗读一下这段话
- 台灯播报"今天天气不错"
- 来段顺口溜
- 播报绕口令
- 帮我念一下这段文字
- 台灯讲个故事
- 播放"小白兔白又白"
- 说个笑话

## 其他控制
- 音量调到最小
- 音量调大一点
- 自适应光源打开
- 自适应光源关闭
- 按键背光关掉
- 按键背光打开
- 入座感应关掉
- 离座感应关掉
- 聚光追踪打开
- 聚光追踪关闭

## 语音 / 拍照
- 播放“你好”
- 让龙虾播放86776656644775
- 拍一张照发给我
- 再拍一张并存下来

## 坐姿 / 学习报告
- 看今天的坐姿检测报告
- 给我最近七天的学习报告
- 看看孩子今天坐姿怎么样
- 帮我整理成一段温和一点的学习建议
- 按微信报告的格式给我一份今天的学习报告
- 用更口语化一点的家长通知风格发我

## 关于首次配置
首次使用时，我会问你手机号，然后往这个手机号发一条验证码短信。收到短信后把验证码告诉我，我来帮你完成登录和设备绑定。整个过程我不会知道验证码长什么样，只是帮你中转一次。

默认情况下，我会直接按**生产环境**来控制这盏灯。

## 关于监控（可选）
如果你希望台灯有任何变化时我主动通知你，可以让我开启监控进程。开启后，我会持续监听台灯状态，一旦发现开关灯、亮度、色温、模式等有任何变化，就会通过飞书推送给你。

要开启监控，只要说"帮我监控台灯"或"开启监控"就好了。

只有在下面几种情况下，我才会额外问你：
- 你明确要求切到测试环境
- 本地没有可用的生产环境缓存
- 当前登录失效，需要重新鉴权
- 你主动要求换设备或换 token

你就算只回复下面这些简写，我也应该能识别：
- `测试`
- `生产`
- `test`
- `prod`

默认规则是：
- **没特别说明，就按生产环境执行**
- **优先复用已经存好的连接信息**
- **不靠聊天记忆，靠本地缓存记住这盏灯**

如果确实需要重新初始化生产环境，我才会再问你：
- 手机号
- 设备 SN

## 关于拍照
如果你说：
- 拍张照
- 再拍一张发给我
- 把它存下来

我会自动做这些事：
1. 让设备拍照
2. 取回图片
3. 尝试直接回图
4. 如你要求，再存到本地目录

## 说明
如果 token 失效，我会再提醒你并重新登录；否则默认一直复用现有 token。
